# MVCGo

MVCGo is API framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling.

## Framework

- Web : GoFiber
- Validation : Go-Ozzo
- Configuration : GoDotEnv
- Database : PostgreSQL
- Hot Reload :Air
